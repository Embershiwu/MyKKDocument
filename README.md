# KK文档仓库


此仓库为 KK社群文档 的文档，大家的文档都可以上传到这里。

文档目前正在更新, 你可以在
https://doc.kkgkd.com
查看。

## 完善文档

如果如果你对于这个文档有好的建议或者补充, 欢迎提 pull request 。

```shell
git clone https://github.com/ThrowPanic/MyKKDocument.git
# clone 源码

cd MyKKDocument

npm install
# 安装依赖

npm run docs:dev
# 开始写作
```
